Lead Management Dashboard
Overview

A mini CRM-style Lead Management Dashboard built as part of a take-home assignment.
The application demonstrates lead listing, filtering, pagination, analytics, and basic authentication.

Features

Login & Logout (basic authentication using localStorage)

Leads listing with search and status filter

Pagination

Lead details view

Analytics metrics:

Total leads

Converted leads
